## Primary operator discipline

The Aletheos binding profile distinguishes motion, admission, return, and receipt.

A is the local motion operator. It records admissible one-step relation inside a declared apparatus or reference core.

M is the admission operator. It records which relations, sectors, claims, certificate fields, addresses, or edges are admitted into the active witness register.

M^T is the return operator. It maps admitted incidence back toward the source side for comparison, verification, and closure.

Q is the receipt operator. It records the overlap, closure, or witness relation produced by admission followed by return.

The primary receipt law is:

    Q = M M^T

An Aletheos-bound object is not trusted merely because it is present. It must pass through admission and return as a reproducible receipt.

In the Thalean reference core, where A is the adjacency operator of G15 isomorphic to L(Petersen), the receipt operator satisfies:

    Q = M M^T = A^3 + 2A^2 + 2I

This reference identity is not required to hold for every possible B32K use.

It is the reference witness showing how local motion, incidence admission, and receipt overlap can close into an exact finite law.

## Sovereign certificate consequence

A sovereign certificate is not valid merely because it exists in storage.

It is valid only when it is normalized, witnessed, authorized, admitted, returned, receipted, and not revoked.

No Q, no trust.
