# Aletheos operator registry 001

Timestamp: 2026-07-08T15:01:47

This note records the candidate operator families Aletheos expects to bind later into the B32K canonical alphabet.

This is not a final canonical index assignment.

The registry says what must be accounted for. It does not yet freeze where those symbols live in B32K.

## Families

Apparatus operators govern trust-bearing transitions.

Modal operators classify the status of a claim.

Assembly operators describe low-level execution and memory actions.

Logical operators evaluate predicates and schema conditions.

Witness-math operators describe passage, return, and receipt.

## Boundary

English labels are advisory.

ASCII tokens are advisory until ratification.

B32K index assignment is the future authoritative symbol identity.

No operator is reliance-bearing merely because it appears in this registry.

Authority requires B32K index binding, schema validation, receipt, and ratification.

## Keeper

The glyph is not the operator.

The word is not the operator.

The index carries the symbol.

The receipt admits the meaning.
