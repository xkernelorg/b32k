# King James Version source corpus

This directory is reserved for the King James Version source text used as the first training corpus for the B32K Concordance Ledger.

Corpus role:

    source text
    phrase extraction
    structural concordance training
    cross-grammar lookup generation

Boundary:

    This is not a doctrine engine.
    This is not a hidden-code claim.
    This is not a physics claim.
    This is a structural translation dataset.
